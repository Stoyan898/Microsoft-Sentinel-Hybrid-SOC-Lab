# 📊 SOC Monitoring Dashboard – Microsoft Sentinel

🚀 Custom SOC monitoring workbook created in Microsoft Sentinel to visualize security telemetry, incidents, and detection activity.

---

## 🎯 What This Dashboard Demonstrates

- SOC monitoring visualization
- Failed login monitoring
- PowerShell abuse visibility
- Sysmon process telemetry
- Sentinel incident overview
- KQL-based workbook creation
- Threat monitoring workflows

---

## 🧰 Technologies Used

- Microsoft Sentinel
- Workbooks
- KQL (Kusto Query Language)
- Sysmon
- Azure Arc
- Windows Security Events

---

## 🏗️ Dashboard Components

### 1. Sentinel Incidents Overview

Displays:
- Total incidents
- Incident severity
- Active alerts
