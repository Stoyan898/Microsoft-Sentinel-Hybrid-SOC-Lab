Microsoft Sentinel Hybrid SOC Lab

This project demonstrates the implementation of a hybrid SOC lab using Microsoft Sentinel and Azure Arc to monitor a locally hosted Windows VM.

The lab focuses on:

Windows Security Event monitoring
Failed login detection
SIEM telemetry ingestion
Azure Arc onboarding
KQL-based investigations

The environment was built using low-cost resources and local virtualization to simulate enterprise SOC workflows.
